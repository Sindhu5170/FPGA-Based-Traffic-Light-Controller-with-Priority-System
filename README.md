# FPGA Traffic Light Controller with Priority System

## Description
This project implements a traffic light controller using VHDL. It follows a finite state machine (FSM) approach to control traffic signals for two roads (North and East).

An emergency input overrides normal operation and gives priority to emergency vehicles.

## Features
- FSM-based traffic control
- Red, Yellow, Green light sequence
- Emergency override mode
- Clock-based timing control

## Tools Used
- VHDL
- ModelSim / Vivado

## Applications
- Smart traffic systems
- FPGA-based automation
- Embedded control systems
